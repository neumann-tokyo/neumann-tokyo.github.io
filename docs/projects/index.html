自社サービス

インターンシップ制度

  • 情報系の高専生や大学生などを対象としたインターンシップ制度を設けております
  • google classroom を活用した完全オンライン体制で実施します
  • Clojure や Flutter を用いたオリジナルドキュメントを使用します

Word Penne

  • 言語学習向けの単語帳アプリ
  • 単語を登録してクイズを解くことで単語を覚えることができます。
  • 使用技術: ClojureScript, Firebaseなど
