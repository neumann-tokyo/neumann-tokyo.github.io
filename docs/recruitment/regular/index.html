業務内容

  • 基本的に業務はすべてテレワークで行う
  • Clojure, Ruby, TypeScript, Dart(Flutter) などのプログラミング言語を用いた Web サービスの構築
  • 上記による受託開発および自社サービスの開発
  • アジャイル開発手法を用いる
  • 海外の案件を行う可能性もあります
  • GitHub, AWS, GCP などのクラウドサービスを用いた開発が多い
  • インターン生などの若手エンジニアの教育
  • 日本人以外とコミュニケーションする機会があるためその場合は英語を使います

必要なスキル・経験

必須

  • Clojure, Ruby, TypeScript, Dart(Flutter) などのプログラミング言語を用いたWebサービスの構築スキル
  • バックエンド、フロントエンド両方作れること

あればなお可

  • Web 系のソフトウェア開発経験 2 年以上
  • Java, PHP などのプログラミング言語を用いた Web サービスの構築スキル
  • スマートフォンアプリケーションの開発スキル
  • 基本情報技術者以上の資格
  • 日常会話レベルの英語力
