## 企業理念

リモートワークにより場所や時間に囚われない仕事をし、ITの力により人々の生活を豊かにする。

### 社名の由来

コンピュータの父と呼ばれるジョン・フォン・ノイマン先生に由来しています。

### リモートワークとは

リモートワーク（またはテレワーク）は会社に出社せず自宅などの好きな場所で仕事をすることです。
仕事をする側は好きな環境で仕事をすることで集中力が高まります。
仕事を発注する側にとってはオフィスの大きさや場所に関係なく発注することができます。

#### 2013年4月〜2016年7月 株式会社永和システムマネジメント

永和システムマネジメントには新卒で入社し、主にRailsを用いた受託開発を3年経験しました。
プロジェクトの概要を下記に記します。

* AWS を用いた C to C Webサービスの構築
* 社内管理ツールパッケージの商品開発支援
* 就職支援 Web サービスの構築。新規開発案件で開発当初から1年半ほど関わりました。エンジニア、リーダーの両方を担当しました。

#### 2016年8月〜2018年8月 株式会社グピポ

知人と二人で起業し、主にRailsを用いた受託開発をおこないました。

* スマートフォンアプリのAPIサーバーの開発
* ECサイトのスマートフォンアプリのAPIサーバーの開発支援
* 就職活動生向け学力試験サービスの開発。開発だけでなくデザインも担当しました。
* 建築部品のECサイト開発。新規開発で設計・開発・運用をおこないました。

#### 2017年8月〜2019年8月 フリーランス

株式会社グピポで仕事をしつつ、副業でフリーランスとしても受託開発の仕事を行いました。

* 岩手県における鳥獣被害対策IoT端末の開発
* ECサイトのサーバー開発支援
* Go言語による B to B 社内管理サービスのパッケージの開発支援

IT以外の事業:

* シェアハウス運営
* イラスト制作
* 技術書同人誌執筆

#### 2019年9月　合同会社ノイマン

今までのエンジニアとしての経験を活かし、さらなる事業拡大のためフリーランスから法人化を行いました。
引き続き受託開発を行いつつ、技術書同人誌の執筆やイラスト制作などの分野でも活動していく予定です。
